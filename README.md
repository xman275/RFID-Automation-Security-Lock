**Description**: Engineered an embedded security system that utilizes Radio Frequency Identification (RFID) to manage physical access control through automated locking mechanisms.

**Tech Stack:** C++, Arduino/ESP32, MFRC522 RFID Module, EEPROM, Servo Control.

**Security Implementation:** Developed logic to cross-reference unique UID tags against an internal database stored in EEPROM, ensuring persistence of authorized credentials even after power loss.

**Authentication Logic:** Integrated a multi-stage authentication process featuring real-time status indicators (LED/Buzzer) and fail-safe protocols to prevent unauthorized access attempts.

**Hardware Integration:** Calibrated high-torque servo motors with precise pulse-width modulation (PWM) to ensure reliable mechanical operation of the locking deadbolt.
